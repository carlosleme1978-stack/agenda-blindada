Política de Privacidade

Última atualização: 30/01/2026

A Agenda Blindada respeita a privacidade dos seus utilizadores e está comprometida em proteger os dados pessoais tratados através da aplicação.

1. Quem somos

A Agenda Blindada é uma aplicação de agendamento profissional com integração ao WhatsApp, utilizada por múltiplas empresas (clientes), cada uma com os seus próprios dados e números.

2. Dados que recolhemos

Podemos recolher e processar os seguintes dados:

  • Nome, telefone e informações de contacto fornecidas pelos clientes finais
  • Dados de agendamento (data, hora, serviço, histórico)
  • Mensagens enviadas via WhatsApp (confirmações e lembretes)
  • Dados técnicos básicos (logs, IP, eventos de sistema)

3. Como utilizamos os dados

Os dados são utilizados exclusivamente para:

  • Gerir agendamentos
  • Enviar confirmações e lembretes via WhatsApp
  • Garantir o funcionamento técnico e a segurança do sistema
  • Cumprir obrigações legais

4. Isolamento e segurança dos dados

Cada empresa que utiliza a Agenda Blindada possui um ambiente isolado. Os dados não são partilhados entre empresas.

Utilizamos boas práticas de segurança, incluindo autenticação, controlo de acessos, isolamento lógico e armazenamento seguro.

5. WhatsApp e serviços de terceiros

A aplicação integra-se com a API oficial do WhatsApp. As mensagens são processadas de acordo com as políticas do WhatsApp e da Meta.

Também utilizamos serviços de infraestrutura (ex.: cloud hosting e base de dados) apenas para o funcionamento do sistema.

6. Retenção de dados

Os dados são mantidos apenas pelo tempo necessário para a finalidade do serviço ou conforme exigido por lei.

7. Direitos do utilizador

O utilizador pode solicitar:

  • Acesso aos seus dados
  • Correção ou atualização
  • Eliminação, quando aplicável
